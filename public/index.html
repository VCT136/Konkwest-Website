Konkwest is a turn based, grand strategy game in which you can finish a multiplayer campaign in a single session.

Features

  • Cross-platform multiplayer
  • AI for non-player countries
  • Custom countries
  • Diplomacy system
  • Industry system
  • Linguistics system
  • Mouse-only playable*
  • Over 1500 places
  • Play as any country
  • Rename countries & more
  • 4 Scenarios
  • Simultaneous turns
  • Spectate mode
  • Supports touchscreens
  • Terrain types
*Except to enter names.
